Question:1 What are the differences between var,let and const.

-- Let: It allows the user to define BOTH GLOBAL/LOCAL SCOPE VARIABLE. If you use Let in a function, then it means the user is defining a local variable but if the user is defining a variable with Let outside the function, then the user is defining a global variable.
--Var: It allows the user to define ONLY GLOBAL SCOPE VARIABLE irrespective of the fact that where the variable has been defined.
-- Const: It’s quite similar to Let with the only difference that once the variable has been defined then its value cannot be changed. Like if the user tries to change the value of the const variable, then they will get an error.

Question:3 What are the differences between Map,foreach,filter and find.

--map: The map function always returns the new array with the same length as the original.
--forEach: The forEach method allows you to run a function for every element of the array. forEach doesn’t return anything.
-- filter: filter works like the find function, but instead of returning the first matched item, filter will return the array of matched items or an empty array if nothing matched.
--find:The find function returns the first item that matches the callback function, or undefined if there is nothing that matches.
